Cara Main

Kuasai permainan dengan langkah-langkah sederhana ini

Langkah 1
01

Lempar Dadu

Mulai giliran Anda dengan melempar dadu. Anda perlu mendapat angka 6 untuk mengeluarkan token dari rumah Anda. Jika Anda mendapat angka 6, Anda mendapat giliran ekstra! Gunakan ini secara strategis untuk memajukan token Anda dengan cepat atau mengeluarkan beberapa token.

Tips Pro:

Selalu coba keluarkan keempat token secepat mungkin untuk lebih banyak opsi pergerakan.

Langkah 2
02

Pindahkan Token Anda

Setelah melempar, pindahkan salah satu token Anda sesuai jumlah ruang yang ditunjukkan dalam arah searah jarum jam. Jika Anda mendarat di ruang yang ditempati token lawan, token mereka akan dikirim kembali ke rumah mereka!

Tips Pro:

Coba jaga token Anda di tempat aman (bintang berwarna) untuk mencegah mereka dikirim kembali ke rumah.

Langkah 3
03

Masuk Jalur Rumah

Setelah token Anda berkeliling papan, ia masuk ke jalur rumah berwarna Anda. Anda harus melempar angka yang tepat untuk mencapai pusat. Jika Anda mendapat angka terlalu tinggi, Anda tidak bisa memindahkan token tersebut.

Tips Pro:

Posisikan beberapa token dekat jalur rumah Anda untuk meningkatkan peluang mendapatkan token ke rumah.

Langkah 4
04

Menang Permainan

Pemain pertama yang berhasil membawa keempat token ke pusat memenangkan permainan! Kemenangan membutuhkan strategi dan sedikit keberuntungan. Apakah Anda bisa mengalahkan lawan Anda?

Tips Pro:

Seimbangkan antara serangan (mengirim lawan pulang) dan pertahanan (memajukan token Anda) untuk peluang menang terbaik.
